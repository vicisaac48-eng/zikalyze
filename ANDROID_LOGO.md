## 游닀 How to Download

### Method 1: Direct Download (Recommended)
1. Click any of the download links above
2. Right-click on the image in your browser
3. Select **"Save Image As..."**
4. Choose where to save the file

### Method 2: Using wget/curl (Command Line)
```bash
# Download the main 512x512 logo
wget https://raw.githubusercontent.com/vicisaac48-eng/zikalyze/main/src/assets/zikalyze-logo.png

# Or using curl
curl -O https://raw.githubusercontent.com/vicisaac48-eng/zikalyze/main/src/assets/zikalyze-logo.png
```

### Method 3: Clone the Repository
```bash
# Clone the entire repository to get all files
git clone https://github.com/vicisaac48-eng/zikalyze.git
cd zikalyze

# Logo files are located at:
# - src/assets/zikalyze-logo.png (main logo)
# - public/pwa-*.png (PWA icons)
# - android/app/src/main/res/mipmap-*/ (Android icons)
```
